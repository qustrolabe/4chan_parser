# 4chan_parser

## Usage

Parse single thread downloading all attachment files in it:
```bash
./parser.py -b pol -t 12345678
```

Parse whole board
```bash
./parser.py -b pol
```

Parse whole board + archive
```bash
./parser.py -b pol -a
```

Parse whole board downloading only preview thumbnails
```bash
./parser.py -b pol -p
```
